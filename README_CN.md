## 支持 :

1. 滚动歌词

2. 逐字歌词

3. LRC(逐字,逐行,ESLyric), ASS, SRT 歌词文件运行时解析

4. 音乐渐进渐出

5. 音乐歌单

6. 音乐封面 KMeans 算法主题色计算

7. 音频可视化

8. 播放器拓展

9. 点击歌词播放

10. 歌词文件导入

11. dmusicpak 支持

---

## 模块

| 模块名称                | 模块描述             |
| ----------------------- | -------------------- |
| DreamMusicPlayer        | 音乐播放器核心实现   |
| DreamMusicPlayerEditor  | 音乐播放器编辑器实现 |
| DreamMusicPlayerLyric   | 歌词文件解析实现     |
| DreamMusicPlayerNetwork | 音乐网络实现         |
| DreamMusicPlayerUMG     | 音乐播放器 UMG 实现  |
