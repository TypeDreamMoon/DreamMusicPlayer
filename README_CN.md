# 梦月音乐播放器

---

### [ENGLISH](README.md)

## 开发者 : [DreamMoon](https://github.com/TypeDreamMoon)

## 支持引擎版本 : 5.x (需自行构建)

---

## 支持 :

1. 滚动歌词

2. 歌词文件读取

3. 音乐渐进渐出

4. 音乐歌单

5. 音乐封面主题色计算

6. 音频可视化

7. 模块化

8. 点击歌词播放

9. ...以后可能支持网络歌曲播放
