## 支持 :

1. 滚动歌词

2. 歌词文件读取

3. 音乐渐进渐出

4. 音乐歌单

5. 音乐封面主题色计算

6. 音频可视化

7. 模块化

8. 点击歌词播放
9. 逐字歌词
10. 歌词文件解析(lrc`WordByWord``LineByLine``ESLyric`, ass, srt)

9. ...以后可能支持网络歌曲播放
